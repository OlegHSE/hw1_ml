# Домашняя работа 1

## Что было сделано:
1. EDA датасета для предсказания стоимости автомобиля
2. Обработка признаков для обучения модели
3. Обучение модели регрессии для предсказания стоимости автомобиля

## Результаты:
Была обучение **Ridge регрессия** на вещественных + категориальных признаках.

**Метрики**:
- `MSE train`: 87252912795.2
- `R2 train`: 0.7
- `MSE test`: 108035968355.3
- `R2 test`: 0.7

Наибольший прирост в качестве дало добавление категориальных признаков по сравнению с качеством только на вещественных признаках.

## Streamlit-приложение:
Чтобы запустить приложение с моделью для предсказания стоимости автомобиля воспользуйтесь следующей командой:
```bash
streamlit run app.py
```

В приложении удалось реализовать:
- Визуализацию вещественных признаков обучающего датасета. В дальнейшем можно добавить визуализацию категориальных признаков.
- Визуализация обученной весов модели.
- Поле для ввода CSV файла с данными для получения предскания.
- Поле для ручного ввода данных для получения предсказания.

**Важно**: 
Чтобы получить предсказания по CSV файлу воспользуйтесь файлом `models/X_test_cat.csv`
